# openAI_gym_breakout_game

Este repositorio contiene pruebas de algoritmos de Deep Reinforcement Learning en el entorno Atari de OpenAI Gym.

**Entorno:** BreakoutNoFrameskip-v4  
**Algoritmo:** PPO  
**Optimizador:** AdamW  
**Función de activación:** ReLU
