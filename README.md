# openAI_gym_breakout_game

Testing some Deep Reinforcement Learning algorithms in Atari OpenAI gym framework.

**Environment:** BreakoutNoFrameskip-v4  
**Algo:** PPO  
**Optimizer:** AdamW  
**Activation fn:** ReLU
